FAQ

How do I convert a chart to image?

Please use RenderTargetBitmap Class.

How do I process a chart that is not on the current screen?

This is not supported yet, keep an eye on #415.

How do I zoom or pan progromatically?

It's as easy as setting the Axis.MinValue and Axis.MaxValue properties, if you need more information please see Data Pagination article.

Why are my animations ugly?

Reference types are easier to track than value types for the library. If the example you are following is plotting a reference type point, like ObservableValue class and you are reproducing it but plotting double values instead, then you might get a different animation.

Why is my chart animation performing slow?

Give LiveCharts.Geared a try! It's an independent package that boost the performance of the LiveCharts library. When LiveCharts was started, my intention was to create beautiful charts, performance was never the target. When the popularity of the library went up, I was forced to add this package so that the library was a full solution.
