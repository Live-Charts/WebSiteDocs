Customizing Tooltips

By default every chart that requires a Tooltip or Legend, initializes a new instance of DefaultLegend or DefaultTooltip.

Customizing defaults

You can customize a little about these classes, for example the background color, bullet size, or orientation, as shown in the next code snippet, 

cartesianChart1.DataTooltip.BulletSize = 20;
cartesianChart1.DataTooltip.Background = Brushes.Red;

You can also set the selection mode of your tooltip. For example, with the next code we force our tooltip to display only the point that was hovered. 

cartesianChart1.DataTooltip.SelectionMode = LiveCharts.TooltipSelectionMode.OnlySender;

From Scratch

The previous block works when you need to customize the appearance of your chart components, but how about modifying the way the data is displayed in the tooltip, or showing extra properties in your tooltips?

Sadly there is no native way to do this in WinForms. Since LiveCharts.Winforms actually is a wrapper for LiveCharts.Wpf, you must define WPF UserControls to make this work, it is really easy and you don't really need to know too much about WPF to make this work.

The DefaultTooltip and DefaultLegend classes try to work for all cases, but if you need a specialized component you can easily define your own as a custom UserControl. LiveCharts will inject the data the user needs to see in the tooltip, and you handle how to display this data according to your needs.
